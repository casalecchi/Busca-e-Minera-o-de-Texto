O modelo é gerado no formato de arquivo CSV da seguinte maneira:

Token       ;  Número_Documento(i) ; Número_Documento(i+1); ...
Termo(j)    ;         W(i, j)      ;        W(i+1, j)     ; ...
Termo(j+1)  ;        W(i, j+1)     ;       W(i+1, j+1)    ; ...
    ...     ;           ...        ;           ...        ; ...
